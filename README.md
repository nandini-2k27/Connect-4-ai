# 🎮 Connect 4 AI Game

An interactive Connect 4 game built using Python and Pygame with an AI opponent powered by Minimax and Alpha-Beta pruning.

## 🚀 Features
- 🧠 AI opponent (Minimax + Alpha-Beta)
- 🎮 Player vs AI and Player vs Player modes
- 🎯 Difficulty levels (Easy, Medium, Hard)
- 🎬 Smooth animations
- 🔊 Sound effects
- 📊 Scoreboard tracking
- ⏱️ Timer-based gameplay
- 🗂️ Match history logging

## 🛠️ Tech Stack
- Python
- Pygame
- NumPy

## ▶️ How to Run

```bash
pip install pygame numpy
python main.py
